## 简介
这是一个React+Nest实现的全栈社区项目，持续更新中

 - /frontend前端目录
 - /server后端目录
 - 前后端都是npm run dev启动
## 配置
/server目录下新建一个.env文件，填入配置

```
DB_HOST=xxx
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=xxx
DB_DATABASE=xxx

EMAIL_HOST=xxx
EMAIL=xxx
EMAIL_PASSWORD=xxx
EMAIL_SECRET=xxx

REDIS_HOST=xxx
REDIS_PORT=6379
REDIS_PASSWORD=xxx
```
